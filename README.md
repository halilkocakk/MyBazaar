# 🛒 MyBazaar – Online Shopping Management System

**Java | Object-Oriented Programming | Course Project**

MyBazaar is a Java-based online shopping management system developed using object-oriented programming principles.

This project was developed as part of the **SENG 271** course at Çankaya University.

---

## 🚀 Features

- Different product types
- Role-based user system
- Object-oriented design

---

## 🧠 OOP Concepts Used

- Inheritance  
- Encapsulation  
- Abstraction  
- Polymorphism  

---

## 🛠 Technologies

- Java  
- IntelliJ IDEA
